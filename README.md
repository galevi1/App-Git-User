# Git User

## A web app for searching GitHub users.

### what can you do?
Get info like followers, 5 recent repos, link to profile and more..

### For this project I used:

- React with hooks.

- React Router.

- Context API.

- GitHub API using axios

- CSS, Font Awesome.
